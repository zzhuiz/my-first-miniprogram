# 校园社团活动报名小程序

## 项目介绍

这是一个基于微信小程序原生语法 + 微信云开发的校园社团活动报名小程序，支持学生端和社团管理员端，包含AI内容审核功能。

## 核心功能

### 学生端
- 微信授权登录，绑定学号、姓名、学院、手机号
- 活动列表：按分类 / 时间筛选，显示封面、标题、人数上限、报名状态
- 活动详情：介绍、时间地点、报名表单、报名按钮
- 报名表单：姓名、学号、学院、联系方式、自我介绍 / 备注
- 我的报名：查看待审核 / 已通过 / 已拒绝，可取消报名
- 消息通知：审核结果推送

### 管理员端
- 活动发布：编辑标题、封面、时间地点、人数上限、截止时间
- 报名列表：查看所有报名，支持导出 Excel
- 审核功能：通过 / 拒绝，填写理由
- 数据统计：报名人数、签到人数

### AI 审核核心
- 对报名备注、自我介绍进行 AI 文本审核
- 自动检测敏感词、违规内容、垃圾信息
- 违规自动标红并拦截提交，给出提示
- 正常内容放行，减轻人工审核压力

### 附加功能
- 活动签到：生成二维码，现场扫码签到
- 报名人数实时更新，满员自动关闭报名
- 表单必填项校验，学号 / 手机号格式校验

## 技术栈

- 前端：微信小程序原生（wxml/wxss/js/json）
- 后端：微信云开发（云数据库 + 云函数）
- AI 审核：调用免费文本内容安全接口

## 部署步骤

### 1. 准备工作
- 注册微信小程序账号
- 下载并安装微信开发者工具
- 开通微信云开发服务

### 2. 项目配置
1. 克隆项目到本地
2. 在微信开发者工具中导入项目
3. 配置云开发环境
   - 在 `app.js` 中修改云开发环境 ID
   - 在 `project.config.json` 中配置云函数目录

### 3. 云函数部署
1. 在微信开发者工具中，右键点击云函数目录，选择「上传并部署」
2. 部署以下云函数：
   - `login`：用户登录和信息绑定
   - `activity`：活动管理
   - `apply`：报名管理
   - `ai审核`：文本内容审核
   - `generateQrcode`：生成签到二维码

### 4. 云数据库配置
在云开发控制台中创建以下集合：
- `users`：用户信息
- `activities`：活动信息
- `applies`：报名信息

### 5. 权限配置
在云开发控制台中，配置云函数的权限，确保能够调用相应的 API。

## 测试说明

### 1. 学生端测试
1. 打开小程序，点击「微信授权登录」
2. 绑定个人信息（学号、姓名、学院、手机号）
3. 浏览活动列表，点击活动进入详情页
4. 填写报名表单，提交报名
5. 在「我的」页面查看报名状态

### 2. 管理员端测试
1. 登录小程序，点击「发布活动」
2. 填写活动信息，发布活动
3. 在活动详情页点击「查看报名列表」
4. 审核报名（通过 / 拒绝）
5. 点击「签到」按钮，生成签到二维码
6. 扫描二维码进行签到测试

### 3. AI 审核测试
1. 在报名表单的「自我介绍/备注」中输入敏感词
2. 提交报名，查看审核结果
3. 输入正常内容，测试审核通过的情况

## 注意事项

- 本项目使用微信云开发服务，需要开通相应的服务
- AI 审核功能依赖微信云开发的内容安全接口
- 签到功能需要用户授权获取位置信息
- 请确保云函数已正确部署，否则功能可能无法正常使用
